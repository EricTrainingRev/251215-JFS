# RevaIssue User Stories

## Epic 1: User Authentication & Role Management
- As a user, I want to log in securely using my credentials so that I can access the system with my assigned permissions.
- As a user, I want to log out so that my session ends and my account remains secure.

## Epic 2: Project Management
- As an Admin, I want to create new projects so that my team can start tracking issues for them.
- As an Admin, I want to update project details so that the information stays accurate.
- As an Admin, I want to delete projects so that obsolete projects are removed from the system.

## Epic 3: Issue Tracking
- As a user, I want to view a list of issues so that I can monitor their status and progress.
- As a user, I want to update issue details so that the information reflects the latest changes.
- As a user, I want to see the history of an issue so that I understand its lifecycle.

## Epic 4: Workflow Management
- As a Tester, I want to open new issues so that bugs are reported promptly.
- As a Developer, I want to move issues to “In Progress” and “Resolved” so that I can indicate work status.
- As a Tester, I want to close issues after successful testing so that resolved bugs are marked as complete.
- As a Tester, I want to reopen issues if the fix fails so that unresolved problems are addressed.

## Epic 5: Search & Filtering
- As a user, I want to filter issues so that I can quickly find relevant issues.
- As a user, I want to search for issues by keywords so that I can locate specific items efficiently.

## Epic 6: Audit Logging
- As an Admin, I want to view audit logs of state changes so that I can track who made changes and when.
- As a user, I want logs to be automatically generated for each transition so that the system maintains accountability.
